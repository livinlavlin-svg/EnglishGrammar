# English Grammar Trainer — Present Perfect

Интерактивный тренажёр для отработки Present Perfect на русском языке.

## О проекте

Тренажёр помогает изучающим английский язык закрепить Present Perfect через практические упражнения. Подходит для самостоятельной подготовки к экзаменам и ежедневной практики.

## Возможности

- Упражнения на образование форм Present Perfect
- Задания на выбор между Past Simple и Present Perfect
- Заполнение пропусков в предложениях
- Мгновенная проверка ответов
- Подсказки и объяснение ошибок

## Технологии

- HTML / CSS / JavaScript

## Запуск

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/livinlavlin-svg/EnglishGrammar.git
   ```
2. Открой `index.html` в браузере.
